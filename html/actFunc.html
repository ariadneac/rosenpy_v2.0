actFunc.acos(module, x, derivative=False)[source]

Activation function - arc cosine, element-wise.

Parameters:

  • xp (str) – CuPy/Numpy module. This parameter is set at the time of initialization of the NeuralNetwork class.

  • x (array_like) – Input array.

  • derivative (bool, optional) – The default is False.

Returns:

It defines whether what will be returned will be the activation function (feedforward) or its derivative (backpropagation).
