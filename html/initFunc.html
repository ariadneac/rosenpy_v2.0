initFunc.glorot_normal(module, rows, cols, i=0)[source]

Initializes a complex matrix using the Glorot normal initialization method.

Parameters:

  • module (str) – CuPy/Numpy module. This parameter is set at the time of initialization of the NeuralNetwork class.

  • rows (int) – The number of rows in the matrix.

  • cols (int) – The number of columns in the matrix.

Returns:

A complex matrix of size (rows, cols) initialized using the Glorot normal initialization method.

Return type:

array-like
